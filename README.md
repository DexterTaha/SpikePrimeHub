# SpikePrimeHub
Technical info about LEGO Spike Prime Hub

Port pin mapping

Port A (lpf2 id 0)  
id1 > Pin.board = PD7  
id2 > Pin.board = PD8  
pwm_pin1 > Pin.board = PE9  
pwm_pin2 > Pin.board = PE11  

Port B (lpf2 id 1)  
id1 > Pin.board = PD9    
id2 > Pin.board = PD10  
pwm_pin1 > Pin.board = PE13  
pwm_pin2 > Pin.board = PE14  

Port C (lpf2 id 2)  
id1 > Pin.board = PD11  
id2 > Pin.board = PE4   
pwm_pin1 > Pin.board = PB6  
pwm_pin2 > Pin.board = PB7  

Port D (lpf2 id 3)  
id1 > Pin.board = PC15  
id2 > Pin.board = PC14  
pwm_pin1 > Pin.board = PB8  
pwm_pin2 > Pin.board = PB9  

Port E (lpf2 id 4)  
id1 > Pin.board = PC13  
id2 > Pin.board = PE12  
pwm_pin1 > Pin.board = PC6  
pwm_pin2 > Pin.board = PC7  

Port F (lpf2 id 5)  
id1 > Pin.board = PC11  
id2 > Pin.board = PE6  
pwm_pin1 > Pin.board = PC8  
pwm_pin2 > in.board = PB1
